## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-org/local-llm-examples.git
   cd local-llm-examples
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Pull the required models:
   ```bash
   npm run models:pull
   ```
